## ðŸ¥—  Implementation

<Steps>
  <Step title="Get the extension from file url">
    Use the `Extract Matching Text` column to get the file extension with regex.
  </Step>
  <Step title="Create an interactive viewer iframe template">
    Use a template column to replace `YOUR_FILE_LINK_HERE` placeholder in the iframe template with your file URL.

    Next, use an `HTML to URL` column to create an URL you can use with Web Embed.
  </Step>
  <Step title="[Optional] Add an alternative static table view">
    Use `Experimental Code` column to create a static HTML from the file (for production uses, please fork the repo).

    Next, use an `HTML to URL` column to create an URL you can use with Web Embed.
  </Step>
  <Step title="Add conditional display">
    Add an `IF Then Else` column to control which URL column makes its way into the Web Embed component based on the file type and user choice.
  </Step>
</Steps>
